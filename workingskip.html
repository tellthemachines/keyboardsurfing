the working skip link

The fix for this bug is very simple: just add the tabindex attribute to the <main> element, preferably giving it a negative value so it is removed from the tab order but still focusable.

<main id="main-content" tabindex="-1">

Now the skip link works on all browsers. But wait! If you're on Chrome, you may have noticed a blue focus ring around the main content section when you tabbed to it. Whoops. That doesn't look awesome. But it's pretty easy to fix.
